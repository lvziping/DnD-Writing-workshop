# D&D 写作工坊

沉浸写作 · 润色改写 · D&D 冒险 — 纯前端单页应用

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 简介

D&D 写作工坊是一款将创意写作与 D&D 桌游冒险结合的在线写作工具。你可以在沉浸式编辑器中专注写作，使用 LLM 进行多风格文本改写，同时与四位 D&D 伙伴角色互动，在写作间隙开启掷骰冒险。

## 功能特色

- **富文本编辑器**：支持 Markdown 语法的所见即所得编辑，浮动格式工具栏，深色/浅色主题切换，专注模式
- **LLM 润色改写**：提供文艺、幽默、润色、简洁、辛辣五种风格，一键改写选中文本
- **D&D 冒险伙伴**：四位个性鲜明的角色陪你写作
  - 🎵 缪斯 · 吟游诗人 — 灵感与文案
  - ⚔️ 铁锤 · 战士 — 结构与节奏
  - 🔮 星墨 · 法师 — 逻辑与深度
  - 🌿 松果 · 德鲁伊 — 情感与氛围
- **D20 冒险一刻**：5 回合掷骰冒险，DC 难度判定，LLM 生成叙事，冒险编年史自动归档
- **存档系统**：多文档管理，快速保存/另存为，冒险进度随文档绑定
- **导出与备份**：PDF 导出，全量数据备份/恢复

## 快速开始

1. 打开 [在线地址](https://dnd-writing-workshop.vercel.app)（部署后替换为实际链接）
2. 点击「新的故事」开始写作
3. 首次使用会自动弹出「冒险百科」引导你配置 API

## API 配置

本应用需要 LLM API 支持，支持以下平台：

| 平台 | 是否免费 | 备注 |
|------|---------|------|
| 硅基流动 | 有免费额度 | 推荐新手使用 |
| Groq | 免费 | 海外平台，速度快 |
| DeepSeek | 按量付费 | 性价比高 |
| OpenAI | 按量付费 | GPT-4o |
| Claude | 按量付费 | Anthropic |

在应用内点击「⚙️ API」按钮，选择平台预设，填入你的 API Key 即可。

## 技术栈

- 纯 HTML + CSS + JavaScript，单文件架构，无需构建工具
- 所有数据存储在浏览器 localStorage，不经过任何第三方服务器
- 兼容 OpenAI API 格式 + Anthropic Claude API

## 隐私说明

你的写作内容和 API Key 仅保存在浏览器本地，不会上传到任何服务器。LLM 请求直接从浏览器发送到你配置的 API 服务商。
